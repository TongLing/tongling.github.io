Paper 1:DESIGN AND IMPLEMENTATION OF ONLINE 3-DIMENSION VIEWER SUPPORTING MULTIPLE FILE FORMAT

Abstract: 3-demension files can be opened currently only in specific design software, which is hard to be viewed in web browsers. To solve the inconvenience of viewing online and transforming file format, the author proposes a technic which can make 3-demension files readable in websites. Besides, the article also explores the methods of displaying 3D model in browser as well as the vital technics of multiple type of file formats transformation. The result demonstrates that the platform can successfully transform 3D file formats within a few seconds as well as presents 3D models online.

Paper 2:THE PROGRESS OF DATA MINING IN SOFTWARE ENGINEERING

Abstract: According to the actual circumstances showing in many software, the technique of data mining has gradually been applied to many fields, analyzing and processing huge data successfully. A number of valuable information has been extracted towards people, which creates benefits both to individuals and corporations. Based on this, the paper investigated reasons why the technology has been widely used, aiming at the challenges in the process of data mining to deepen the understanding of the technology.
