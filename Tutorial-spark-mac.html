Tutorial for Spark Environment Configuration on Mac OS System

This tutorial will help you install and configure all required environment for python and spark on Mac OS system. After setting it up, your system should be able to run spark program using python programming language in PyCharm.

In this tutorial, we use PyCharm as a efficient IDE to configure all environment. PyCharm will combine python with spark so that you can code using Spark in this IDE.

1.Installing PySpark 
MacOS usually comes with Python installed. But to make sure that you have Python installed, open the terminal.

And run the following command.
$ python --version
If this command returns a version number such as 
Python 2.7.10
or any other version number, that means Python is already installed in your system. It also means that you already have access to easy_install. If python is not exist in your system, be sure to visit www.python.org/downloads/ to install the latest version for Mac OS. 
Now, all you have to do is run the following command. You may enter your MacOS login password after enter this command when you use “sudo”. The password would not show explicitly on the Terminal.
$ sudo easy_install pip
After the pip installation is finished, with Spark 2.2 you can simply type
$ pip install pyspark
to install all required package for Pyspark. Then type 
$ pyspark
you may see
      ____              __
     / __/__  ___ _____/ /__
    _\ \/ _ \/ _ `/ __/  '_/
   /__ / .__/\_,_/_/ /_/\_\   version 2.2.0
      /_/
    Now the Pyspark configuration should be done in your system. To test the pyspark installation, type the following python code on your command:

    >>> words = sc.parallelize(["scala","java","hadoop","spark","akka"])
    >>> words.count()
    >>> 5

    If you see the output 5 is on your command, the Pyspark configuration should be all set.

2.Installing PyCharm and py4j
    First, Download PyCharm from this website: https://www.jetbrains.com/pycharm/download/
    To Install pycharm on your machine, just unpack the pycharm.dmg and drag the installation package into the application folder.
    Then, use the following command to Install py4j. 
    $ sudo pip install py4j
3.Pycharm Configuration
    Go through the following below code and add the appropriate SPARK_HOME directory and PYTHONPATH (Spark Path) folder to successfully import the Apache Spark modules. SPARK_HOME is where you place the spark packege. PYTHONPATH is the path for Pyspark.
    The directory of your SPARK_HOME should be: "/usr/local/Cellar/apache-spark/2.2.0"
    Your pyspark PYTHONPATH should be: “/usr/local/Cellar/apache-spark/2.2.0/bin”

    Sometimes the directory SPARK_HOME and PYTHONPATH might not exist, however it may move to other paths due to the change of your operating system and the pyspark version. You may need to figure out the correct path.
    Create a new Python project (“File->NewProject->Create”, then right click the folder on the left sidebar, choose “new->Python file->OK”), then copy and paste the following code to the board.
import os
import sys

# Path for spark source folder
os.environ['SPARK_HOME']= "SPARK_HOME"

# Append pyspark  to Python Path
sys.path.append("PYTHONPATH")
try:
    from pyspark import SparkContext
    from pyspark import SparkConf
    print ("Successfully imported Spark Modules")
except ImportError as e:
    print("Can not import Spark Modules", e)
    sys.exit(1)

    After executing the code, it should print “Successfully imported Spark Modules”
    Then, you have the correct path for SPARK HOME and PYTHONPATH. Open PyCharm and create a new Project. in the menu choose “Run” ->“Edit Configurations” ->“Environment variables” 
    
    Add to path in the “environment variables”. we need to add “SPARK_HOME” and “PYTHONPATH” to the list.
    Note: your directory path should be different from mine. Make sure two path values is exist and correct on your system.
