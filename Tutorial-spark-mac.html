Tutorial for Spark Environment Installation on Mac OS System

This tutorial will help you install and configure all required environment for python and spark on Mac OS system. After setting it up, your system should be able to run spark program using python programming language in a PyCharm software.

In this tutorial, we use PyCharm as a efficient IDE to program our work. PyCharm will combine python and spark package so that you can do everything about spark on this IDE.

1.Installing PySpark 
MacOS usually comes with Python installed. But to make sure that you have Python installed, open the terminal.

and run the following command.
$ python --version
If this command returns a version number such as 
Python 2.7.10
or any other version number, that means Python exists. Which also means that you already have access to easy_install. If python is not exist on your system, visit www.python.org/downloads/ to install the latest version for Mac OS. 
Now, all you have to do is run the following command. you may enter your MacOS login password after enter this command. The password would not show explicitly on the Terminal.
$ sudo easy_install pip
after the pip installation is finished, With Spark 2.2 you can simply do 
$ pip install pyspark
to install all required prerequisite package on pyspark. type
$ pyspark
>>> words.count()
you may see
      ____              __
     / __/__  ___ _____/ /__
    _\ \/ _ \/ _ `/ __/  '_/
   /__ / .__/\_,_/_/ /_/\_\   version 2.2.0
      /_/
    Now the Pyspark configuration should be done. to test the pyspark installation, run the following python code:

    >>> words = sc.parallelize(["scala","java","hadoop","spark","akka"])
    >>> words.count()
    >>> 5
    If you see the output 5 is on your command, the Pyspark configuration is all set.. 

2.installing PyCharm and py4j
    My operating system: Mac OS 10.13.2
    First, Downloading PyCharm from website: https://www.jetbrains.com/pycharm/download/ 。
    Installing pycharm: unpack the pycharm.dmg and drag the installation package into the application folder
    Installing py4j：
    $ sudo pip install py4j
3.Pycharm Configuration
    Go through the following below code and add the appropriate SPARK HOME directory and PYSARK folder to successfully import the Apache Spark modules. 
    the directory of your spark home should be: "/usr/local/Cellar/apache-spark/2.2.0"
    Your pyspark python path should be: “/usr/local/Cellar/apache-spark/2.2.0/bin"

import os
import sys
# Path for spark source folder
os.environ['SPARK_HOME']=“SPARK_HOME"

# Append pyspark  to Python Path
sys.path.append(“PYTHONPATH")
 
try:
    from pyspark import SparkContext
    from pyspark import SparkConf
 
   print ("Successfully imported Spark Modules")
 
except ImportError as e:
    print ("Can not import Spark Modules", e)
    sys.exit(1)

    After executing the code it should print “Successfully imported Spark Modules”
    Open PyCharm and create a new Project. in the menu choose “Run” ->“Edit Configurations” ->“Environment variables” 
    
    Add to path in the “environment variables”. we need to add “SPARK_HOME” and “PYTHONPATH” the value is spark root directory. 
    Note: your directory path should be different from mine. Make sure the path values is exist and correct on your system. 

4.Testing Pycharm

    Copy and paste the following PySpark code:

from pyspark import SparkContext

logFile = "SPARK_HOME” + “/README.md"
sc = SparkContext("local","Simple App")
logData = sc.textFile(logFile).cache()

numAs = logData.filter(lambda s: 'a' in s).count()
numBs = logData.filter(lambda s: 'b' in s).count()

print("Lines with a: %i, lines with b: %i"%(numAs, numBs))

    if the output shows as follows,  environment for spark and python is all set.
Lines with a: 58, lines with b: 26
